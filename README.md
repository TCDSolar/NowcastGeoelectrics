# NowcastGeoelectrics
Scripts for nowcasting geoelectric fields across Ireland are currently being optimised. Scripts adapted from previous work by Campanya et al 2018 (see https://github.com/joancampanya/Compute_GeoElectric_Fields)

# Packages Required:
numpy
matplotlib
pandas
datetime
scipy
time 
urllib
geopy
Basemap
ffmpy
cv2 
os

#Optional Packages

osgeo
seaborn
shapely
fiona 
gdal
